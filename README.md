## Info

- os: Artix Linux
- wm: dwm
- term: st
- launcher: dmenu
- lockscreen: slock
- bar: dwmblocks-async
- shell: mksh/dash
- editor: NvChad (Neovim)
- notifs: dunst
- compositor: compfy (I have this off most of the time)
